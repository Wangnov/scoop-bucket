# scoop-bucket

Scoop bucket for Wangnov apps and CLI tools.

```powershell
scoop bucket add wangnov https://github.com/Wangnov/scoop-bucket
```

## Codex App Manager

Install, update, and uninstall the official Codex desktop app — with built-in, China-reachable self-update.

```powershell
scoop install wangnov/codex-app-manager
```

The manifest is kept in sync with the latest [Codex App Manager](https://github.com/Wangnov/Codex-App-Manager) release automatically (see `.github/workflows/bump-codex-app-manager.yml`).
